What is Deep Learning?

Deep Learning is a machine learning technique where a computer learns by example. The results can exceed human performance. The computer continually learns using data and algorithms. An example of how deep learning works is available at mathworks.com


We generate an estimated 2.6 quintillion bytes of data every day. Deep learning uses this data and algorithms to learn. This allows machines to solve complex problems such as facial recognition, medecine, shopping, and entertainment. See more deep learning example at Forbes.com
